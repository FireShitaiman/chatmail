Chatmail

メールをチャットのように。Gmailをチャット風UIで使えるデスクトップアプリ。

サービス概要

ChatmailはGmailをチャットスレッド形式で表示・操作できるElectronデスクトップアプリケーションです。 メールスレッドをチャットUIに変換し、返信・転送・作成を一画面で行えます。 オープンソース・無料で利用できます。

主な機能

チャット風スレッド表示

Gmailのスレッドをチャット形式で見やすく表示します。

AI返信支援

Claude / OpenAI APIを使った返信文生成・トーン調整（フォーマル・カジュアル・短縮）。

スパムフィルター

ルールベース＋AIによるスパム検出・自動振り分け。

タスク管理

メールをタスクとして登録し、メモを付けて管理できます。

アドレス補完

送受信履歴から宛先を自動補完します。

ダーク/ライトテーマ

環境に合わせてテーマを切り替えられます。

動作環境

  • Windows / macOS / Linux
  • Node.js 18 以上
  • Google Cloud プロジェクト（Gmail API有効化・OAuthクライアントID）
  • AI機能を使う場合：Claude APIキー または OpenAI互換APIキー（任意）
